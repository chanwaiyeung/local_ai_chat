## Pass Criteria

- Present facts must be answered from retrieved chunks, not general knowledge.
- Absent facts must return a clear "not found in the document" answer.
- Each citation link must open the document viewer.
- The viewer must scroll to the cited chunk and highlight it.
- Multi-turn follow-ups must stay grounded in the active document.
- Synonym rewrites must not be rejected only because wording differs from the document.
- The chunks sent to the LLM and the citations shown under the answer must come from the same reranked hit list.
- `flutter analyze` must report no errors.

## Regression Notes

- Score-only filtering is not enough for small local embedding models because unrelated chunks can still receive high cosine scores.
- When citation precision degrades, inspect chunk size, top-k, and whether the cited chunk contains the actual answer text.
- If many valid questions are rejected, review the keyword stopword list and consider a stronger embedding model.
